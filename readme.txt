Display links (blogroll) as dropdown select menu

== Description ==

Display links (blogroll) as dropdown select menu with multi option. Visit the <a href="http://blog.casanova.vn/wordpress-blogroll-select-menu-dropdown-widget/" target="_blank">blogroll select menu dropdown widget</a> for more information about the plugin or comment it if you want to support. See more features below

= Features =

The options page allows you to choose from a range of options including:

* How many links to display
* Class css for your select box
* Orderby & Order
* Display links in list of categories (separated by comma) 
* Include or exclude links
* Option open in new window or current window

== Installation ==

1. Download the plugin.
2. Upload it to the plugins folder of your blog.
3. Goto the Settings --> Appeanrance --> Widget and drag drop Blogroll Dropdown to your sidebar
